	Aplikasi ini berfungsi sebagai absensi kelas
	Cara Menggunakan:
		[1]Pilih Mulai
		[2]Pilih Menu:
			[1]DATA, untuk memasukkan data kelas
			[2]ABSENSI, untuk mengisi absensi
	Menu DATA:
		[1]Tambah Data, untuk menambah data kelas
		[2]Lihat Data, untuk melihat daftar data kelas
		[3]Cari Data, untuk mencari info data kelas
		[4]Ubah Data, untuk menghapus maupun merubah informasi data kelas
		[5]Kembali ke menu utama");
	Menu ABSENSI:
		[1]Absensi, untuk mengisi absensi kelas
		[2]Lihat Absensi, untuk melihat daftar hadir kelas
		[3]Kembali ke menu utama
	Format Nama = Namadepan_Namatengah_Namabelakang
	Contoh: Joseph_Elroy_Manurung
